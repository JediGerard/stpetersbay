Menu Migration Tool

This tool uploads menu items from sample_menu.json to Firestore.
Note: For Phase 2, menus are still loaded from sample_menu.json locally. This migration is optional and prepares for future Phase 3 (Admin CMS).

Instructions:

  1. Click "Load Menu from JSON" to preview the menu data
  2. Review the items that will be uploaded
  3. Click "Upload to Firestore" to migrate the data
  4. This is a ONE-TIME operation. Don't run it multiple times.
